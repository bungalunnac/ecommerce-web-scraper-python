# Automated E-Commerce Data Scraper
A Python-based web scraping tool built to extract product data dynamically from e-commerce websites.

## Features
- Automated HTML parsing using BeautifulSoup and Requests.
- Extracts product titles and pricing instantly.
- Exports clean, structured data into CSV format for market analysis.

## Tech Stack
- Python
- BeautifulSoup4
- Requests
- Pandas
